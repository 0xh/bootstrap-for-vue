#### Tooltip
It uses [bootstrap tooltip](https://v4-alpha.getbootstrap.com/components/tooltips/#usage) to render a tooltip. 

`v-tooltip`

##### Example

**Tooltip:**
````vue
<div id="app">
  <a href="#" v-tooltip="Tooltip text here">Link</a>
</div>

<script>
new Vue({
  el: '#app',
})
</script>
````
